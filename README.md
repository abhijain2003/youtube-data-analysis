# youtube-data-analysis
Here, I analyze real-time youtube channel data, using api, visualization and interactive dashboard, and searching most watched videos, help youtuber to identify what kind of videos people like to watch
After having a brief idea on what kind of videos people love to watch most help youtuber to focus more on creating content which is more likely to such videos.

## How to use it.
clone the git repository and open jupyter notebook to view the file and then press run all in notebook.
